Overview
========

The MISP DXL Python Service exposes access to the
`MISP REST APIs <https://misp.gitbooks.io/misp-book/content/automation/#automation-api>`_
via the `Data Exchange Layer <http://www.mcafee.com/us/solutions/data-exchange-layer.aspx>`_
(DXL) fabric. The service also provides support for forwarding
`MISP ZeroMQ <https://misp.gitbooks.io/misp-book/content/misp-zmq/>`_ message
notifications to the DXL fabric.
